From Python Website

Python has efficient high-level data structures and a simple but effective approach to object-oriented programming. Python’s elegant syntax and dynamic typing (need not declare type, automatically inferred during runtime example int in c++), together with its interpreted nature (not translated to machine code instead interpreter runs line by line, this gives advantage to work with large datasets as you don't need to restart like compiled languages), make it an ideal language for scripting and rapid application development in many areas on most platforms.
